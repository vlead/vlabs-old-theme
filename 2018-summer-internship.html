2018 Virtual Labs Summer Internship Program

May 14, 2018 to July 13, 2018

Interns/Projects/Experience

Ananya Arun

Project Title: Heap Sort: Algorithm and Complexity

As a part of the VLEAD summer internship, i built a virtual lab for heap sort experiment. I applied pedagogy principles along with literate style programming to render a working app in two months.Building the experiment involved writing a story board as an org file which is exported to html by a parser.This intermediate html grammar was transformed to an abstract syntax tree or the catalogs which mapped to the artifacts i built. Artifacts were API's using javascript libraries like D3 and SVG's. A linker goes through the catalog service and renders the experiment in a shell with all images and interactive JS artifacts.

Experience: In VLEAD I had the opportunity to discuss and interact with mentors,make frequent presentations and knowledge transfer sessions and now i can confidently say i have understood the entire cycle in the building of software.Overall it was a summer well spent and a great learning experience. Looking forward for my experiment to be hosted and put into use.
